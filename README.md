## 📖 About

This repository contains various **Shadowrocket** modules and configuration files for iOS.

Supported file types:

* `.sgmodule`
* `.conf`

---

# 🚀 How to Use

### Method 1 — Import `.sgmodule`

1. Open this repository.
2. Navigate to the desired **`.sgmodule`** file.
3. Click **Raw**.
4. Copy the **Raw URL**.
5. Open **Shadowrocket**.
6. Go to:

```
Modules
    ↓
+
    ↓
Download from URL
```

7. Paste the Raw URL.
8. Save and enable the module.

---

### Method 2 — Import `.conf`

1. Open this repository.
2. Navigate to the desired **`.conf`** file.
3. Click **Raw**.
4. Copy the **Raw URL**.
5. Open **Shadowrocket**.
6. Go to:

```
Configuration
    ↓
+
    ↓
Download from URL
```

7. Paste the Raw URL.
8. Save and use the configuration.
